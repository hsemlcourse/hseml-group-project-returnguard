## Описание задачи

Проект посвящён анализу микроданных выборочного наблюдения доходов населения Тульской области за 2023 год. На основе записей о домохозяйствах с 42 социально-экономическими характеристиками решаются две задачи:

**Задача 1 (Регрессия):** Предсказание среднедушевого дохода домохозяйства (`doxodn`)

**Задача 2 (Классификация):** Отнесение домохозяйства к группе с доходом выше третьего квартиля

**Датасет:** Микроданные выборочного наблюдения доходов населения, Тульская область (код 71), 2023 год. 3 803 домохозяйства, 42 признака.

**Признаки модели:** `chlico` (число лиц), `chdet` (число детей), `rasress` (располагаемые ресурсы), `rasq` (среднедушевые ресурсы), `potras` (расход на потребление)

**Целевые метрики:**
- Регрессия: R², RMSE, MAE
- Классификация: Accuracy, ROC-AUC, F1-score

## Запуск

### Локально
```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
pip install -r requirements.txt
pytest tests/test.py -v
```

### Docker
```bash
# Тесты
docker compose run tests

# Jupyter Lab на http://localhost:8888
docker compose up notebook

# FastAPI + Streamlit (после обучения моделей)
docker compose up api streamlit
```

### API (FastAPI)

После запуска `docker compose up api` или `uvicorn app.main:app --reload`:

```bash
# Предсказание
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"chlico": 3, "chdet": 1, "rasress": 200000, "rasq": 70000, "potras": 150000}'

# Документация
open http://localhost:8000/docs
```

### Streamlit

После `docker compose up api streamlit` — интерфейс доступен на `http://localhost:8501`.
